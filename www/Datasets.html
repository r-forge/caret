Other Interesting Datasets

 
 

Chemistry


  1. •There are several computational chemistry data sets in the QSARdata package on CRAN.


  1. •Karthikeyan et al (2005) presented data where they used chemical descriptors to model the melting point of compounds (i.e. transition from solid to liquid state). They assembled 4401 compounds: 4126 for model training and 275 compounds as a final validation set. They calculated 2D and 3D MOE chemical descriptors. Descriptions of these variables can be found at http://www.chemcomp.com/journal/descr.htm. They used a combination of PCA and neural networks to create models. The data are in tab delimited format in this zip file.


  1. •Tetko et al. (2001) and Huuskonen (2000) investigated a set of compounds with corresponding experimental solubility values using complex sets of descriptors. They used linear regression and neural network models to estimate the relationship between chemical structure and solubility. This RData file contains 1267 compounds and a set of more understandable descriptors that fall into one of three groups: 208 binary “fingerprints” that indicate the presence or absence of a particular chemical sub–structure, 16 count descriptors, such as the number of bonds or the number of Bromine atoms and 4 continuous descriptors, such as molecular weight or surface area. The data were split using random sampling into a training set (n = 951) and test set (n = 316).


  1. •Lombardo et al (2006) predict human volume of distribution using 384 compounds. These data can be found in an sdf file format here.


Business Analytics


  1. •SGI has several data sets on their homepage for MLC++ software at http://www.sgi.com/tech/mlc/. One data set can be used to predict telecom customer churn based on information about their account. The data are here.


  1. •Larose (2006) contains several data sets, including one for direct marketing where a clothing company is interested in offering promotions by mail to its customers. Using existing customer data on shopping habits, they wish to predict who would respond (i.e. the two classes and "responders" and "non-responders"). Those data are here.
